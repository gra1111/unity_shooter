# Unity Shooter

A fast-paced first-person shooter (FPS) project developed in Unity.

## Controls
- **WASD**: Movement
- **Mouse**: Look around
- **Left Click**: Shoot
- **Shift**: Sprint
- **ESC**: Pause
- **R**: Reload
- **G**: Throw grenades

## Objective
Survive for as long as possible by defeating various enemies, each featuring unique combat mechanics and attack patterns.
