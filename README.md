# Unity Shooter

Proyecto de shooter en primera persona desarrollado en Unity.

## Controles
- WASD: Movimiento
- Ratón: Mirar
- Click izquierdo: Disparar
- Shift: Correr
- ESC: Pausa
- R: recargar
- G: lanzar granadas

## Objetivo
Sobrevive el mayor tiempo posible derrotando dsitintos enemigos con distintas mecanicas y ataques cada uno.
